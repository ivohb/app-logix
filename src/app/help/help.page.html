Esta documentação apresneta o escopo do projeto, descrevendo, de forma resumida, os módulos pelos quais o APP é composto. Informações mais detalhadas encontram-se no help de cada módulo, que poderam ser visualizados, a partir da tela principal do aplicativo (tela Home).
- EDI Cliente:
O objetivo desse módulo é permitir aos clientes da Ethos a elaboração de listas de programação de compra para um determinado produto, listas essas que, posteriormente, serão integradas com a carteira de pedidos do sistema ERP da Ethos.
- EDI Kanabn:
O objetivo desse módulo é permitir aos usuários da Ethos a elaboração de listas de programação de venda para um determinado item kanban, listas essas que, posteriormente, serão integradas com a carteira de pedidos do sistema ERP da Ethos.
- Inventário:
O objetivo desse módulo é possibilitar aos usuários o lançamento de listas com os itens que devem entrar em processo de inventário, possibilitar o lançamento de listas com as contagens dos itens e finalizar o inventário dos itens enviando-os juntamente com suas respectivas contagens para o módulo de suprimento do ERP.
