{{'OBJETIVO'|translate}}
O objetivo desse módulo é permitir aos clientes da Ethos a elaboração de listas de programação de compra para um determinado produto, listas essas que, posteriormente, serão integradas com a carteira de pedidos do sistema ERP da Ethos. Suas funcionalidades são:
- Iniciar:
Permite ao cliente iniciar uma nova lista de programação para um determinado produto.
- Consultar:
Permite ao cliente consultar uma lista de programação previamente iniciada, para visualização e/ou alteração nos dados.
- Integrar:
processa a lista de programação previamente selecionada, enviando seus dados diretamente para a carteira de pedidos do ERP.
Obs: para a integração da lista de programação com a carteira de pedidos do ERP, é necessário que, no ERP, haja um pedido aberto para o produto. Se não houver, o usuário Ethos deverá providenciar a inclusão de um pedido (via programa padrão do ERP) para o produto, e depois executar a integração da lista novamente.
